Este programa Python é um simples gerador de frases de motivação que exibe uma frase motivacional aleatória lida de um arquivo de texto. Ele também inclui a funcionalidade de limpar a tela antes de mostrar a frase.

Funcionalidades:

	1.	Leitura de Frases de Motivação: O programa lê frases motivacionais de um arquivo de texto chamado ‘frases.txt’. O arquivo deve conter frases, cada uma em uma nova linha.
	2.	Escolha Aleatória de Frase: Utilizando a biblioteca random, o programa escolhe uma frase aleatória do arquivo para ser exibida.
	3.	Limpeza de Tela: Antes de mostrar a frase, o programa verifica o sistema operacional em que está sendo executado (macOS, Linux ou Windows) e utiliza o comando apropriado para limpar a tela.
	4.	Exibição da Frase Motivacional: Após limpar a tela, o programa exibe a frase motivacional escolhida no terminal ou prompt de comando.

Uso:

Para executar o programa, basta rodar o código em um ambiente Python compatível. Certifique-se de que o arquivo ‘frases.txt’ esteja no mesmo diretório em que o programa está sendo executado. O programa exibirá uma frase motivacional aleatória após limpar a tela.

Esta aplicação é uma maneira simples de obter uma dose rápida de motivação e também demonstra como usar comandos para limpar a tela no terminal, tornando a exibição das frases mais agradável e legível.
